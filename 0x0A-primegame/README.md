## Prime Game

### Description
This is a function that determines the winner of a prime game.

### Prototype
`def isWinner(x, nums)`
where `x` is the number of rounds and `nums` is a list of `n` non-negative integers.

### Example:
```python
>>> x = 5
>>> nums = [0, 1, 2, 3, 4, 5]
>>> isWinner(x, nums)
Ben
```
